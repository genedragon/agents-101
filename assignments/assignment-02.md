# Assignment 2: My Workspace, My First Files

**Due:** End of session 2 (finish at home if needed)
**Estimated time:** 20–30 minutes
**Format:** Do it in Kiro IDE on your own laptop. Bring the result next week.

---

## Goal

Prove you can open a workspace and get the agent to create and change files for you — and that you can name the parts of the Kiro window.

---

## Part 1 — Label your window (5 min)

In your notebook (paper or digital), draw a rough rectangle for the Kiro window and label these four parts. Write one sentence for each explaining what it does:

- File tree
- Editor
- Terminal
- Agent chat

---

## Part 2 — Make three files by asking (15 min)

In your `my-first-workspace` folder, ask the agent to create these three files. **You are not allowed to type the contents yourself — you have to ask the agent.**

| File | Ask the agent to... |
|---|---|
| `about-me.md` | Write 3 fun facts about you |
| `hobby.md` | Describe one hobby and why you like it |
| `goals.md` | List 2 things you want to build in this club |

After each one, **open the file and read it.** Did it get it right? If not, ask the agent to fix it.

---

## Part 3 — Change a file (5 min)

Ask the agent to **add one more line** to `about-me.md` — a fourth fun fact. Watch the file update in the editor.

Write down in your notebook: *did you touch the file directly, or did the agent?*

---

## Part 4 — Stretch (optional)

Ask the agent to create a file `mystery.md` and tell it: *"put something in here you think I'd find interesting."* See what it chooses. Was it a good guess? Write one line about whether the agent "knew" you or just guessed.

---

## How this gets graded

"Did you try?" — not right-or-wrong. We're looking for:
- Your four labels make sense
- All three files exist and were made by asking the agent
- You noticed that *asking* is different from *typing it yourself*
