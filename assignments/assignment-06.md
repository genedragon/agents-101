# Assignment 6: Ship It — Go Live on the Internet

**Due:** End of session 6 (finish at home if needed)
**Estimated time:** 25–35 minutes
**Format:** GitHub + your project. This is the big one.

---

## Goal

Get your project onto GitHub and live on a public URL — then share it.

---

## Part 1 — Get on GitHub (15 min)

With the agent's help:

> "Help me put this project on GitHub."

Let it walk you through creating a repo, committing, and pushing. Read the permission prompts as you go.

Checklist:
- [ ] Repo created on GitHub
- [ ] My project files are in the repo (check on github.com)

---

## Part 2 — Turn on Pages (10 min)

On GitHub: **Settings → Pages → Deploy from a branch → `main` / root.**

Wait 1–2 minutes for it to build, then open your live URL.

- [ ] My live URL opens in a browser
- [ ] It opens on my **phone** too

Write your live URL in your notebook: `https://__________.github.io/__________/`

---

## Part 3 — Share it (5 min)

Text or send your live URL to **one person** — a friend, a parent, a sibling. Write down what they said.

This is real. Something you built is on the internet now.

---

## Part 4 — Honesty check (5 min)

Look at the course sample site ("Sprig") for the shape. Then make sure your own site is **honest**: if an agent helped you build it, say so somewhere (a footer line is enough). We never pass off agent work as fully human-made.

Add one line to your site, like: *"Built by [your name] with help from an AI agent, in the Agents 101 club."*

---

## How this gets graded

"Did you try?" We're looking for:
- A working live GitHub Pages URL
- You shared it with a real person
- An honesty line on your site

---

## Bring next week

- Your live URL (working!)
- One thing about your project that's **broken or annoying** — we fix bugs next week

---

## If you're stuck

Auth and Pages settings are the two sticky spots — totally normal. If push fails, ask the agent to use HTTPS + browser sign-in, or upload files through GitHub's website as a fallback. If Pages shows a 404, double-check it's set to `main` / root and give it another minute. Ask in the club chat with your repo link.
